# Neovim configs based on NvChad

## Installation

1. **(Optional) Backup your current nvim configs:**
```bash
mv ~/.config/nvim/ ~/.config/nvim.bak/
```

2. **Clone repo into .config:**
```bash
git clone https://github.com/say8hi/nvim-dotfiles.git ~/.config/nvim && nvim
```
3. **Run install plugins command inside nvim:**

```
:MasonInstallAll
```
